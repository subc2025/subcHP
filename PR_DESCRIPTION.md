## ⚙️ 設定手順
1. GitHubリポジトリの「Settings」→「Pages」に移動
2. Source を「Deploy from a branch」に設定  
3. Branch を「main」、フォルダを「/ (root)」に設定
4. 「Save」をクリック
5. 数分後にサイトが公開されます

この設定により、永続的な固定URLでサイトにアクセスできます。
